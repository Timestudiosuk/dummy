== Description ==

Allow your clients to visually submit feedback and requests directly on their own website, while keeping it all in one place.
Saving you and your clients countless hours by getting straight to the point and getting things done!

== Installation ==

1. Go to https://atarim.io/ > purchase plugin

2. Go to Plugins > Add New.
Click Upload Plugin to display the WordPress Plugin upload field.

3. Click Choose File to navigate your local file directory.

4. Select the WordPress Plugin zip archive you wish to upload and install.

5. Activate the plugin.

6. Go to Atarim > Atarim Settings in Admin. Select "Settings" tab, check "Enable Atarim on this website" option and click "Save Changes" button.

7. Go to Frontend and enjoy adding comments.
